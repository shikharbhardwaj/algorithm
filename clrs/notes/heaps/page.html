Heap
This is a short note regarding the use of the heap data structure in implementing an efficient priority queue.

A heap is a nearly complete binary tree stored in a well defined order of priority.
The order of priority is decided by the presence of keys associated with each node.

Heap properties
The heap can have two kinds of properties : The max-heap property and the min-heap property.

Max-heap : a heap has the max heap property if, for each node, the key of the children is <= the key of the node.
Min-heap : a heap has the min heap property if, for each node, the key of the childeren is >= the key of the node.

These properties can be established in an unordered heap using the methods `build-max-heap`
